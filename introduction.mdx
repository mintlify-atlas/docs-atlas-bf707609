## What is Opik?

Opik is an open-source platform designed to streamline the entire lifecycle of LLM applications. Built by [Comet](https://www.comet.com), Opik empowers developers to evaluate, test, monitor, and optimize their models and agentic systems from development through production.

<CardGroup cols={2}>
  <Card title="Comprehensive Observability" icon="chart-line">
    Deep tracing of LLM calls, conversation logging, and agent activity with detailed context
  </Card>
  <Card title="Advanced Evaluation" icon="flask">
    Robust prompt evaluation, LLM-as-a-judge metrics, and experiment management
  </Card>
  <Card title="Production-Ready" icon="server">
    Scalable monitoring dashboards and online evaluation rules for production environments
  </Card>
  <Card title="Agent Optimizer" icon="robot">
    Dedicated SDK and optimizers to enhance prompts and agent workflows

## Key Features

<AccordionGroup>
  <Accordion title="Development & Tracing" icon="code">
    - Track all LLM calls and traces with detailed context during development and in production
    - 50+ integrations with frameworks including OpenAI, Anthropic, LangChain, LlamaIndex, Google ADK, Autogen, and more
    - Annotate traces and spans with feedback scores via the Python SDK or UI
    - Experiment with prompts and models in the Prompt Playground
  </Accordion>

  <Accordion title="Evaluation & Testing" icon="vial">
    - Automate LLM application evaluation with datasets and experiments
    - Leverage powerful LLM-as-a-judge metrics for hallucination detection, moderation, and RAG assessment
    - Integrate evaluations into CI/CD pipelines with PyTest integration
    - Custom metrics for domain-specific evaluation needs
  </Accordion>

  <Accordion title="Production Monitoring & Optimization" icon="gauge-high">
    - Log high volumes of production traces (40M+ traces/day)
    - Monitor feedback scores, trace counts, and token usage over time in dashboards
    - Utilize online evaluation rules with LLM-as-a-Judge metrics to identify production issues
    - Leverage Opik Agent Optimizer and Guardrails to continuously improve and secure your applications
  </Accordion>
</AccordionGroup>
