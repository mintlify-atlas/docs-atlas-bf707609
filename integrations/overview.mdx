## Integration Categories

### What are Agent Frameworks?

Agent frameworks provide structured approaches to building AI agents that can:
- Make decisions and take actions
- Use tools and call functions
- Maintain state and memory
- Coordinate multiple agents

Examples: Google ADK, OpenAI Agents, CrewAI, Pydantic AI, Smolagents

### What are LLM Frameworks?

LLM frameworks help you build applications with language models:
- RAG (Retrieval Augmented Generation) pipelines
- Prompt management and chaining
- Document processing and indexing
- Memory and conversation management

Examples: LangChain, LlamaIndex, Haystack, DSPy

### What are Gateways?

Gateways provide unified interfaces to multiple LLM providers:
- Single API for multiple providers
- Load balancing and failover
- Caching and rate limiting
- Cost optimization

Examples: LiteLLM, OpenRouter, Portkey, Kong AI Gateway

### What are Validation Tools?

Validation tools ensure quality and safety of LLM outputs:
- Input/output validation
- Guardrails and safety checks
- Schema enforcement
- Content moderation

Examples: Guardrails AI
