## Installation

Install the Opik Python SDK:

<CodeGroup>
```bash pip
pip install opik
```

```bash uv
uv pip install opik
```
</CodeGroup>

## Configuration

Configure the SDK:

```bash
opik configure
```

For Opik Cloud, you'll need your API key from [comet.com](https://www.comet.com).

For self-hosted, see the [installation guide](/installation).

## Log Your First Trace

```python
import opik

@opik.track
def my_llm_function(question: str) -> str:
    return f"Answer to: {question}"

result = my_llm_function("What is Opik?")
opik.flush_tracker()
```

View your trace at [comet.com](https://www.comet.com).

## Add LLM Integration

```python
import opik
from openai import OpenAI

client = opik.integrations.openai.track_openai(OpenAI())

@opik.track
def ask_question(question: str) -> str:
    response = client.chat.completions.create(
        model="gpt-4",
        messages=[{"role": "user", "content": question}]
    )
    return response.choices[0].message.content

answer = ask_question("What is Opik?")
opik.flush_tracker()
```

## Run an Evaluation

```python
from opik.evaluation import evaluate
from opik.evaluation.metrics import Equals

def task(item):
    return {"output": "Paris"}

result = evaluate(
    dataset=opik.Opik().get_dataset("my-dataset"),
    task=task,
    scoring_metrics=[Equals()]
)
```
