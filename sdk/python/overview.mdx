The Opik Python SDK requires:
- Python 3.10 or higher
- An Opik server instance (cloud or self-hosted)

## Quick Start

Here's a simple example to get started with Opik:

```python
import opik

# Configure Opik
opik.configure(
    api_key="YOUR_API_KEY",
    workspace="YOUR_WORKSPACE"
)

# Track a function
@opik.track
def my_llm_function(user_question: str) -> str:
    # Your LLM logic here
    return f"Answer to: {user_question}"

# Call the function - it will be automatically traced
result = my_llm_function("What is Opik?")

# Flush to ensure all data is sent
opik.flush_tracker()
```

## Core Components

### Opik Client

The `Opik` class is the main interface for interacting with the Opik platform:

```python
from opik import Opik

client = Opik(
    project_name="my-project",
    workspace="my-workspace"
)
```

See the [Client reference](/sdk/python/client) for detailed documentation.

### Tracing

Use the `@track` decorator to automatically log traces and spans:

```python
import opik

@opik.track
def process_query(query: str) -> dict:
    # Your processing logic
    return {"result": "processed"}
```

See the [Tracing guide](/sdk/python/tracing) for more details.

### Evaluation

Evaluate your LLM applications with datasets and metrics:

```python
from opik import evaluate
from opik.evaluation.metrics import Hallucination

results = evaluate(
    dataset=dataset,
    task=my_llm_function,
    scoring_metrics=[Hallucination()],
    experiment_name="my-experiment"
)
```

See the [Evaluation guide](/sdk/python/evaluation) for comprehensive examples.

### Datasets

Create and manage datasets for evaluation:

```python
client = Opik()
dataset = client.create_dataset(name="my-dataset")

dataset.insert([
    {"input": {"question": "What is AI?"}, "expected_output": "..."}
])
```

See the [Datasets guide](/sdk/python/datasets) for more information.

### Experiments

Run experiments to compare different approaches:

```python
from opik import evaluate

results = evaluate(
    experiment_name="gpt-4-test",
    dataset=dataset,
    task=my_task,
    scoring_metrics=[metric1, metric2]
)
```

See the [Experiments guide](/sdk/python/experiments) for detailed examples.

## Dynamic Tracing Control

Control tracing behavior at runtime:

```python
import opik

# Disable tracing globally
opik.set_tracing_active(False)

# Check current state
print(opik.is_tracing_active())  # False

# Re-enable tracing
opik.set_tracing_active(True)

# Reset to default
opik.reset_tracing_to_config_default()
```

This is useful for:
- Performance optimization in high-throughput systems
- Conditional tracing based on user type or request parameters
- Debugging without redeployment
- Implementing sampling strategies
